# boutique_gh
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Este repositório contém o projeto de banco de dados desenvolvido para a disciplina de Banco de Dados, focado em comandos DDL e DML. 

O cenário escolhido é uma boutique de luxo da marca Gabriela Hearst, que opera com coleções de unidades limitadas e um sistema rigoroso de lista de espera.

## O objetivo principal foi planejar, criar e popular um banco de dados relacional que gerenciasse:
+ **Clientes VIP** e seus respectivos níveis.
+ **Peças da Coleção** com seu estoque extremamente limitado.
+ A **Lista de Espera** para itens esgotados ou de pré-venda, com previsão de anos para atendimento.

Ao concluir este projeto, obtive uma compreensão prática da **Linguagem de Definição de Dados (DDL)** para estruturar o banco e da **Linguagem de Manipulação de Dados (DML)** para interagir com os dados.

## Como Executar o Script SQL
1. Instale um SGBD (ex: MySQL).
2. Abra o seu cliente de banco de dados (ex: MySQL Workbench).
3. Execute o arquivo `sql_scripts/boutique_gh.sql`.
4. Os comandos DDL (CREATE DATABASE, CREATE TABLE) e DML (INSERT, UPDATE, DELETE) serão executados em ordem.
